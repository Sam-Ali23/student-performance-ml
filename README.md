# Student Performance Prediction

This project predicts student academic performance using Machine Learning.

## Model
- Algorithm: Random Forest Classifier
- Task: Multi-class classification (Low, Medium, High)

## Features
- Student engagement (raised hands, discussions, announcements)
- Attendance
- Parental involvement
- Learning resources usage

## Explainability
- SHAP was used to explain feature importance and model decisions.

## Deployment
- Streamlit web application (UI)
